# Green Apples Delight

A community-driven encyclopedia documenting green apple recipes, confections, and fictional gaming mods.

## 📁 Repository Structure

* `assets/css/` — Custom MediaWiki styles.
* `assets/images/` — Game sprites and photos.
* `pages/index.md` — Main landing page.
* `pages/apple-pie.md` — Minecraft block data.

## 🚀 Quick Start

```bash
git clone https://github.com
cd green-apple-delight
npm install
npm run dev
```

## 📝 Contribution Guide

* Use standard Markdown headers.
* Include nutritional facts for real food.
* Use tables for item crafting recipes.
* Cite external recipe origins.
